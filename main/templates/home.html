Checkin

A family and task management tool based that allows users to create lovations and to-do lists. User can then use the iOS app to check-in at locations, which will automatically pull up to-do lists that were created for that location and that user. This is based on a Django Rest back-end with an Angular front-end. The iOS app is currently in beta testing.

Designed from the start to be compatible with web, iOS and Android apps

An initial preview is available on JC2DEV.com
