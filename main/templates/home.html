Current Projects

Checkin

A family and task management tool that allows users to create locations and to-do lists. Users can then use the iOS app to check-in at locations, which will automatically pull up to-do lists that were created for that location and that user. This is both a web and iOS app. The web app is based on a Django Rest back-end with an Angular front-end. The iOS app is currently in beta testing.

Designed from the start to be compatible with web, iOS and Android (coming soon) apps

An initial preview of the web app is available on JC2DEV.com

Naturalization app for iOS

A test prep app for those taking the US naturalization test (Citizenship Test). It features study, fast-quiz and full-quiz modes.

Group project - Movie Picker

This was a group project done together with three other developers at Coding Campus

It's a web app that will make choosing a movie between a group of people a much easier and less painful process. Users can add movies to the database and vote for them. All votes will be automatically counted for those users who join a movie-watching event.

Check it out online.

Group project - Recipes

This was another group project that was completed together with three other developers at Coding Campus

It's a recipe repository where a user can store their recipes, record their personal reviews of the recipes and other facts about them. It can also dynamically scale a recipe for more or less servings.
